# Comms

Configuration and context for the `/comms` skill.

## Setup

Uses Beeper's built-in MCP server at `http://localhost:23373/v0/mcp`.

- Enable in Beeper Desktop: Settings → Developers → toggle "Beeper Desktop API"
- Registered with Claude Code via: `claude mcp add beeper http://localhost:23373/v0/mcp -t http -s user`
- Docs: https://developers.beeper.com/desktop-api/mcp
